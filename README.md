# ¡Bienvenidos a tv-app!
### ¿Qué es tv-app?
tv-app es una aplicación para Smart TV que implementa una interfaz adaptada al uso con un mando a distancia. El proyecto permite mostrar contenido en una pantalla de televisión y gestionar la navegación entre las distintas opciones mediante controles específicos para este tipo de dispositivos. Está orientado al desarrollo de aplicaciones para televisores utilizando tecnologías web estándar.
